# Smart Clinic User Stories

### 1. Admin: Manage Doctor Profiles
- **User Story:** As an admin, I want to add, edit, and remove doctor profiles so that the clinic directory is always up to date.
- **Priority:** High
- **Story Points:** 5
- **Acceptance Criteria:**
    - Admin can fill a form with Doctor Name, Specialty, and Contact.
    - System validates that no fields are left empty.
    - Success message appears after a doctor is added.

### 2. Patient: Search for Doctors
- **User Story:** As a patient, I want to search for doctors by specialty so that I can find the right consultant for my health issue.
- **Priority:** High
- **Story Points:** 3
- **Acceptance Criteria:**
    - Search bar filters the doctor list in real-time.
    - System displays specialty and availability for each result.

### 3. Doctor: View Daily Schedule
- **User Story:** As a doctor, I want to view all my appointments for the current day so that I can manage my time effectively.
- **Priority:** Medium
- **Story Points:** 3
- **Acceptance Criteria:**
    - Dashboard shows a list of patient names and appointment times.
    - Records are sorted chronologically.

### 4. Patient: Book Appointment Slot
- **User Story:** As a patient, I want to book a specific time slot so that I don't have to wait in a long queue at the clinic.
- **Priority:** High
- **Story Points:** 8
- **Acceptance Criteria:**
    - User can select a date and an available time.
    - System prevents double-booking of the same slot.
    - Email confirmation is triggered upon booking.

### 5. Admin: Generate Monthly Reports
- **User Story:** As an admin, I want to generate a report of total appointments per month so that I can analyze clinic performance.
- **Priority:** Low
- **Story Points:** 5
- **Acceptance Criteria:**
    - Admin can select a month and year.
    - System displays total patient count and revenue.
